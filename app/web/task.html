Техническое задание разработки проекта для Технологий программирования

Тема: Учёт успеваемости студентов на факультете

Постановка задачи:
Веб-приложение со следующими возможностями:

  • регистрация пользователей и разделение их по ролям (администраторы, преподаватели, пользователи, гости)
  • создание, редактирование, удаление - предметов, оценок

Методы решения задачи:
Будет проведен анализ аналогов.
Будет разработан проект решающий поставленную задачу.
Развертывание на веб-хостинге sprinthost или аналогичном.
Составление отчета о проделанной работе.

Инструментальные средства, использованные для разработки приложения:

  • Библиотеки и фреймворки: Yii2, Bootstrap 3, jQuery
  • Языки: PHP, JavaScript, CSS, HTML
  • СУБД: MySQL и phpMyAdmin

Функционал:
Разделить пользователей на 4 роли:

  1. Администраторы:
    • могут менять статус зарегистрированного пользователя
  2. Преподаватель:
    • могут создавать, редактировать, удалять - предметы, оценки
  3. Пользователи:
    • могут просматривать ведомость (таблицу оценок)
  4. Гости:
    • могут просматривать только основную информацию страницы
